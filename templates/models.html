Cross Validation

Five-fold stratified cross validation was performed to verify model stability and ensure consistent performance across different subsets of the dataset.

Hyperparameter Tuning

GridSearchCV was used to optimize Logistic Regression, Decision Tree, Random Forest and K-Nearest Neighbors, selecting the best configuration before deployment.

Probability Calibration

Sigmoid calibration improved the reliability of predicted probabilities by reducing the Brier Score while preserving classification performance.

Evaluated Machine Learning Algorithms

Logistic Regression

Linear baseline classifier for binary classification.

Decision Tree

Tree-based classifier capable of learning nonlinear decision boundaries.

Random Forest

Ensemble learning algorithm combining multiple decision trees to improve prediction accuracy and reduce overfitting.

K-Nearest Neighbors

Instance-based learning algorithm using neighbouring samples for classification.

Gaussian Naive Bayes

Probabilistic classifier based on Bayes' theorem with Gaussian feature assumptions.

Support Vector Machine

Margin-based classifier that finds the optimal separating hyperplane between classes.
