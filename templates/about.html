About This Project

The Customer Churn Prediction Platform is a Machine Learning based web application developed using Flask and Scikit-learn. It predicts whether a customer is likely to discontinue a company's service and demonstrates a complete end-to-end machine learning workflow from data preprocessing to web deployment.

Project Overview

The platform helps organizations identify customers who are likely to churn using demographic and behavioural information. It supports both single customer prediction through an interactive form and batch prediction using CSV files. Predictions include calibrated probability estimates generated by the deployed machine learning model.

Final Deployed Model

The final deployed model is a Calibrated Random Forest Classifier. It was selected after evaluating six supervised machine learning algorithms using five-fold cross validation, hyperparameter tuning with GridSearchCV and probability calibration using the sigmoid method.
