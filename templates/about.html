About This Project

Overview

This Customer Churn Prediction Platform predicts whether a customer is likely to leave a service using machine learning.

Dataset

The application uses customer demographic and behavioural data, including age, tenure, subscription type, support calls, payment delay, total spend and last interaction.

Machine Learning Models

  • Logistic Regression
  • Decision Tree
  • Random Forest
  • K-Nearest Neighbors
  • Gaussian Naive Bayes
  • Support Vector Machine

Technology Stack

  • Python
  • Flask
  • Pandas
  • NumPy
  • Scikit-learn
  • Matplotlib
  • HTML
  • CSS
