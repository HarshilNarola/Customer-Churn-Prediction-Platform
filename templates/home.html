Customer Churn Prediction Platform

Predict whether a customer is likely to discontinue a service using Machine Learning. The platform supports both individual customer prediction and batch prediction through CSV file uploads using a calibrated Random Forest classifier that achieved over 99.9% accuracy on the evaluation dataset.

Single Prediction Batch Prediction

Project Overview

Customer churn prediction enables organizations to proactively identify customers who are likely to discontinue using a service. This platform combines data preprocessing, machine learning, hyperparameter tuning, five-fold cross validation and probability calibration to deliver reliable churn predictions for both individual customers and large datasets.

Single Prediction

Enter customer information through an interactive form and instantly predict whether the customer is likely to churn along with the calibrated probability.

Batch Prediction

Upload a CSV file containing multiple customer records and download a prediction report with churn labels and calibrated prediction probabilities.

Machine Learning Pipeline

Six supervised learning algorithms were evaluated. A calibrated Random Forest model was selected after hyperparameter tuning, cross validation and probability calibration for deployment.

Prediction Workflow

Input Data → Preprocessing → Calibrated Random Forest → Probability Estimation → Prediction Result
