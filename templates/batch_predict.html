Batch Customer Churn Prediction

Upload a CSV file containing customer information. The deployed Calibrated Random Forest model will predict churn for every customer and generate a downloadable CSV report containing both predictions and calibrated probability scores.

Sample Input File

Download the sample CSV to understand the required file structure before uploading your own dataset.

Required CSV Columns

Your CSV file must contain the following columns exactly as shown below.

Column Name
Age
Gender
Tenure
Usage Frequency
Support Calls
Payment Delay
Subscription Type
Contract Length
Total Spend
Last Interaction

Output File

After processing, the generated CSV file will include:

  • Original customer information
  • Predicted Class (Stay / Churn)
  • Calibrated Churn Probability (%)
Note: The uploaded CSV must use the exact column names shown above. Missing or incorrectly named columns will cause the upload to fail.
